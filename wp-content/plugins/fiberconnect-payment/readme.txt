FiberConnect Payment Plugin for WooCommerce allows instant payment collection via QR Code on your online store without deep integration.

== Description ==

= What makes our plugin special? =

* Quick Installation: Install the plugin to your WordPress online store easily
* Security: Our API infrastructure is verified by bank partners
* 24/7 Reporting: Real-time transaction data lookup

= One-click to enable payment methods on your online store =

* Local payment methods for checkout: Support FPS, Alipay HK & WeChat Pay HK
* Mobile-friendly and fast checkout: Checkout is now just a few taps away on mobile
* Real-time payment status update: Dashboard for staff monitoring sales trend and auto payment confirmation to customer

= Get Started =

* You need to create a [FiberConnect](https://fiberapi.com/en/) account for enabling the plugin.

== Installation ==

Here is the steps to setup FiberConnect for WooCommerce payment plugin:

1. Make sure you have installed WooCommerce. If not, you can find it [here](https://wordpress.org/plugins/woocommerce/).
1. You need to create a [FiberConnect](https://fiberapi.com/en/contact-2/) account for enabling the plugin.
1. Upload the plugin to your WordPress online store or install directly through marketplace.
1. Activate the plugin in your WordPress plugin settings and complete the setup.
1. Go to WooCommerce > Settings > Payments and manage the plugin configurations.
1. Collect payment instantly!
